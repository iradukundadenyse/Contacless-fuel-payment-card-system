# Contactless Fuel Payment Card System

## Project Description
A database system developed using Oracle SQL and PL/SQL to manage contactless fuel payments.

## Objectives
- Manage customers
- Manage payment cards
- Process fuel payments
- Store transaction records
- Provide security and auditing

## Database Tables
1. CUSTOMER
2. PAYMENT_CARD
3. FUEL_TRANSACTION
4. AUDIT_LOG
5. HOLIDAY
