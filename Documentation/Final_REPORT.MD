# Final Report
## Project Title
Development of Contactless Fuel Payment Card System

## Problem Statement
Many fuel stations use cash payments which can cause delays and poor transaction tracking. This project provides a contactless card payment system to make fuel payments faster and more secure.

## Objectives
- Manage customer information
- Manage payment cards
- Process fuel payments
- Store fuel transactions
- Provide auditing and security control

## System Users
- Customer
- Fuel station attendant
- Administrator

## Database Tables
1. CUSTOMER
2. PAYMENT_CARD
3. FUEL_TRANSACTION
4. AUDIT_LOG
5. HOLIDAY

## Technologies Used
- Oracle Database
- SQL
- PL/SQL
- GitHub

## PL/SQL Implementation
- Procedures
- Functions
- Cursors
- Packages
- Triggers
- Exception handling

## Conclusion
The system improves fuel payment speed, security, and transaction management.
